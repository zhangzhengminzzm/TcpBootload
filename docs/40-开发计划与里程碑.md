# 开发计划与里程碑

## M1：基础联通（1 周）

- 完成上位机项目骨架与 UI 主窗体
- 完成 Modbus TCP 连接、读写测试
- 输出设备寄存器映射表（升级相关）

验收标准：
- 可连接目标设备并读到关键状态寄存器

## M2：升级闭环（2 周）

- 完成升级状态机与分包下载
- 完成异常重试、超时处理、日志
- 完成单设备升级闭环验证

验收标准：
- 固件升级成功率达到目标（例如 > 99%）

## M3：工程化（1 周）

- 增加可视化日志筛选与导出
- 增加配置文件（默认超时、包长、重试）
- 打包发布（Windows 可执行）

验收标准：
- 支持现场工程师独立使用并可追溯日志

## M4：增强能力（可选）

- 批量升级（多设备队列）
- 固件签名校验
- 版本回退保护策略
